# Fundamentos de la Inteligencia Artificial

## Propósito de la página
Esta página web tiene como objetivo introducir de forma clara y accesible los conceptos fundamentales de la Inteligencia Artificial (IA). Está dirigida a estudiantes, principiantes y personas interesadas en comprender qué es la IA y sus aplicaciones.

## Semántica HTML5 utilizada
La página emplea etiquetas semánticas modernas:

- `<header>` para el encabezado principal.
- `<nav>` para la navegación interna del sitio.
- `<main>` para el contenido principal.
- `<section>` para separar temas principales.
- `<article>` para contenido independiente.
- `<aside>` para información complementaria.
- `<footer>` para créditos del sitio.

Esto mejora la estructura lógica, SEO y accesibilidad.

## Accesibilidad aplicada
- Todas las imágenes incluyen atributo `alt` descriptivo.
- Formularios con etiquetas `<label>` asociadas correctamente.
- Navegación clara y compatible con teclado.
- Uso de `aria-label` en botones para mayor claridad.
- Colores con contraste adecuado.
